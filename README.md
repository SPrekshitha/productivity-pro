# Productivity Pro

Built using HTML, CSS, and JavaScript.

Productivity Pro is a multi-page web application designed to help users manage daily tasks and events efficiently. It combines a to-do list and a calendar system in a simple and user-friendly interface.

## Live Demo
[View Project](https://sprekshitha.github.io/productivity-pro/)

---

## Features

### To-Do List
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Real-time search to filter tasks

### Calendar
- Select specific dates
- Add events for each date
- Edit and delete events
- Track completion status of events

### Additional Features
- Dark/Light mode toggle
- Data persistence using browser localStorage
